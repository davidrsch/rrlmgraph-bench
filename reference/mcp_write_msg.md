# Write a single JSON-RPC message as a newline-terminated line to the stdin of an MCP server process started by `mcp_start_server()`.

Write a single JSON-RPC message as a newline-terminated line to the
stdin of an MCP server process started by
[`mcp_start_server()`](https://davidrsch.github.io/rrlmgraph-bench/reference/mcp_start_server.md).

## Usage

``` r
mcp_write_msg(proc, msg)
```
