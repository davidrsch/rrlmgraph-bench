# rrlmgraphbench: Benchmark Suite for rrlmgraph

Provides fixture projects and ground-truth datasets for benchmarking the
rrlmgraph package. Fixtures range from a minimal data-science script
project to a medium-sized Shiny application and a small R package,
covering the full spectrum of project types supported by rrlmgraph.
